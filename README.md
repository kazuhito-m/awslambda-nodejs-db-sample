# AWS Lambda and Node.js and DB(PostgreSQL) Sample

+ master : XXX by Wercker

## What's this

以下の組み合わせのサンプル。

- AWS Lambda
- Node.js
- PostgreSQL(pg)
- gulp

使いまわせることを期待して課題に対して解決するよう作っています。

## Requirement

- Node.js : v0.12.7  or Later
- npm : 2.11.3  or Later
- AWS SDK 用の `.aws/credentials` が設定されていること

## Setup for Develop

- `npm install`

## Deploy

- `npm run deploy`
